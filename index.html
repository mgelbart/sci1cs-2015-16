Homework collaboration policy:
You are welcome to discuss assignments with others, but the assignments you submit must be entirely your own work. Here are some things that are OK and not OK:
  • OK: Discussing concepts, or how to approach an assignment with your classmates or others (e.g., "Hey, do you think we need to use a nested loop for this assignment?").
  • OK: Discussing Python syntax with your classmates or others, or on Piazza (e.g., "Hey, what is the command for generating a random number between 0 and 1?")
  • OK: Showing your terminal window to a classmate or Piazza and seeking advice based on the output and/or error message your assignment code produced.
  • OK: Having the instructor or TA look at your code for an assignment and give help or suggestions.
  • OK: Typing code on someone else's laptop in order to demonstrate a concept, so long as this code (or any derivative or modified version of this code) is not later submitted as part of an assignment.
  • NOT OK: Typing code on someone else's computer (or having someone type on yours) and then submitting this code as part of an assignment.
  • NOT OK: Looking at a classmate's code for an assignment (or having a classmate look at yours).
  • NOT OK: Sending/receiving any amount of code for an assignment to/from someone else by email, Facebook, or any other means.
  • NOT OK: Posting assignment solution code on Piazza.
  • NOT OK: Having a discussion with someone about an assignment within the limits of what is allowed by this collaboration policy, but then not listing this person's name in your assignment submission (see "Homework submission instructions" below).
If you are caught cheating, the penalty is typically suspension from the University. Please don't do it! If you are struggling to the point that you are tempted to cheat, please come talk to one of us and we will work through the situation. We are here to support you, but there are 75 of you and sometimes we need your help identifying where and when our support is most needed.

Homework submission instructions:
  1. Make sure there is a comment at the top of EVERY Python file you are submitting, containing:
    • your first name
    • your last name
    • your UBC student number
    • a list of everyone you discussed the assignment with, including classmates (first and last name), TAs, instructors, anyone else
    • (optional) the amount of time you spent working on the program
  2. Make sure the name(s) of your file(s) EXACTLY match the name(s) asked for in the assignment.
  3. Go to www.connect.ubc.ca
  4. Sign in with your CWL
  5. Navigate to Science One Computer Science (not the general Science One page)
  6. Click on the assignment you are submitting
  7. Click on Browse My Computer and attach all relevant files, one by one
  8. Press Submit
Note that you will be unable to sumbit assignments on Connect once the submission deadline has passed.
